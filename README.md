#Emlog 博客程序模板LR_ Concision
#介绍
        1.整体采用蓝灰色调，灰色为主
        2.LOGO采用文字形式
        3.顶部公告调用Emlog后台的博客信息
        4.支持调用二级导航
        5.首页文章缩略图自动调用博文首张图片
        6.文章底部简洁版权声明
        7.评论列表采用楼层和等级显示
        8.所有评论调用Gavatar头像
        9.友链，分类，归档采用两列显示
        10.增加返回顶部功能
        11.导航栏添加社交功能按钮(请自行修改header.php49-59行相关代码)
        12.自定义订阅功能（调用QQ订阅列表）
        13.修改置顶文章图标为大拇指
        14.优化日历功能 
        15.去除微语功能
#使用说明
        1.官方地址：http://www.emlog.net/template/426
        2.点击右侧Download zip下载，后台上传，激活即可使用
        3.仅支持Emlog 5.1.2
